## 1

输出相关：S1-S3 A[i]
反相关：S1-S3 A[i], S1-S2 B[i], S2-S3 A[i], S3-S4 C[i]
真相关：S1-S2 A[i], S3-S4 A[i]

展开二级循环，并不会引入相关

| S   | Instr                     |
| --- | ------------------------- |
| S1  | A[i] = A[i] \* B[i]       |
| S2  | B[i] = A[i] + c           |
| S3  | A[i] = C[i] \* c          |
| S4  | C[i] = D[i] \* A[i]       |
| S1' | A[i+1] = A[i+1] \* B[i+1] |
| S2' | B[i+1] = A[i+1] + c       |
| S3' | A[i+1] = C[i+1] \* c      |
| S4' | C[i+1] = D[i+1] \* A[i+1] |

修改变量名如下

| S   | Instr                |
| --- | -------------------- |
| S1  | A1[i] = A[i] \* B[i] |
| S2  | B1[i] = A1[i] + c    |
| S3  | A2[i] = C[i] \* c    |
| S4  | C[i] = D[i] \* A2[i] |
